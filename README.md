## Deskripsi singkat per-berkas
1. matmul.py - Implementasi piton pake logika perkalian matriks per vektor kolom
2. matmul.c - Translasi matmul.py sehingga menggunakan C; Bisa diparalelisasikan dengan OMP
3. matmul-simpler.c - Implementasi klasik matmul tanpa transpose; Bisa diparalelisasikan dengan OMP
4. matmul-mpi.c - Implementasi Claude perkalian matriks menggunakan MPI (belum di-review)
